## Installation

Install icq-clone with npm

```bash
  npm install icq-clone
  cd icq-clone
```
## Run Locally

Clone the project

```bash
  git clone https://github.com/Bleyobeth/icq-clone
```

Go to the project directory

```bash
  cd icq-clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Features

- adding pictures
- adding files
- session saving 
- real-time communication without delays
